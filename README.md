## SESSL-SSJ: A SESSL Binding for the Stochastic Simulation in Java (SSJ) library

SSJ documentation can be found at: http://www.iro.umontreal.ca/~simardr/ssj/indexe.html

- Scala 2.10.1 (http://scala-lang.org)
